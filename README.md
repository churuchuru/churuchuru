# Churu Churu

## Setup

Install new conda environment
```
conda create --name py3.11 python=3.11 -c conda-forge
```

Install reflex
```
pip install reflex
```

Make new project
```
mkdir my_app_name
cd my_app_name
reflex init
```

## Running

Run application in development mode
```
reflex run
```

Debugging
```
reflex run --loglevel debug
```
